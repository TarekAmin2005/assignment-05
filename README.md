## 1. What is the difference between var, let, and const?
- var is old, can change, function-scoped.  
- let can change, block-scoped.  
- const cannot change, block-scoped.  

## 2. What is the spread operator (...)?
- Spread operator (...) copies or expands arrays or objects.  

## 3. What is the difference between map(), filter(), and forEach()?
- map() changes each item and returns a new array.  
- filter() keeps only items that pass a test, returns new array.  
- forEach() runs a function for each item, doesn’t return anything.  

## 4. What is an arrow function?
- Arrow function is a short way to write a function.  

## 5. What are template literals?
- Template literals are strings with `${}` to insert variables.
